Button

Displays a button or a component that looks like a button. Supports 5 variants, 5 sizes, icon placement, and interactive states.

5 Variants 5 Sizes 4 States

Variants

Use variant to set the visual style. Primary is the highest emphasis action.

<!-- Primary -->
<button class="btn btn-primary btn-md">Primary</button>

<!-- Secondary -->
<button class="btn btn-secondary btn-md">Secondary</button>

<!-- Outline -->
<button class="btn btn-outline btn-md">Outline</button>

<!-- Ghost -->
<button class="btn btn-ghost btn-md">Ghost</button>

<!-- Danger -->
<button class="btn btn-danger btn-md">Danger</button>

Class Description Preview
btn-primary High-emphasis CTA. Brand lime background.
btn-secondary Medium emphasis. Transparent with brand border.
btn-outline Low emphasis. Neutral border, muted text.
btn-ghost Minimal. No border, transparent background.
btn-danger Destructive actions. Red background.
btn-xs / sm / md / lg / xl Height: 28 / 34 / 42 / 50 / 60px
btn-loading Loading state with spinner. Reduces opacity.
btn-icon-* Equal width/height for icon-only buttons.
